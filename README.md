# AgentSmith Desktop

AgentSmith Desktop is the cross-platform desktop companion app for AgentSmith.

V1 scope:

- Sign in to a single AgentSmith deployment
- List the current user's file libraries
- Let the user activate or deactivate local mounts
- Restore activated mounts when the app restarts
- Keep mounts alive while the tray app is running
- Unmount when the app exits
