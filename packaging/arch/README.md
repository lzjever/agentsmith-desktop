# Arch packaging

This directory stores Arch Linux packaging metadata for the internal pilot release.

- `PKGBUILD` and `.SRCINFO` are generated by `npm run package:arch:metadata`
- `npm run package:arch` stages the built desktop binary and produces a `.pkg.tar.zst`
- output packages are written to `packaging/arch/out/dist/`

The current pilot package name is `agentsmith-desktop-bin`.
